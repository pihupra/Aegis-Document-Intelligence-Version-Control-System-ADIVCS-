# Aegis Document Intelligence Version Control System (ADIVCS)

## **Features**

- Create and manage documents with metadata (title, author, tags)
- Add multiple versions for each document
- View differences between versions (Myers/LCS diff)
- Search documents by title or tags
- CLI interface for lightweight access
- Qt5 GUI interface with full functionality
